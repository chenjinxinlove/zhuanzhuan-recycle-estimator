# zhuanzhuan-recycle-estimator

转转回收估价 Skill —— 发送图片或文字，快速获取回收报价。

## 中文版

### name

转转回收估价

### short_description

支持图片和文字输入的二手商品估价 Skill，快速识别商品并给出回收价格范围。

### long_description

转转回收估价是一个二手商品智能估价 Skill。用户可以上传图片、输入文字描述，或图文结合，系统会根据商品品类、品牌、型号及关键属性，给出一个可供参考的回收价格区间。

该 Skill 面向"先看看大概能卖多少"的场景，无需立即下单。即使信息不完整，也会尽可能给出估价区间，并提示补充哪些信息可以让报价更精准。

### sample_prompts

- 这台手机大概还能卖多少钱？
- 帮我估一下这部 iPhone 13
- 我发几张图，你帮我看下这台笔记本值多少
- 128G 的，屏幕有一点划痕，大概什么价位？
- 这台平板能卖多少钱？
- 不知道具体型号，你先根据图片帮我估个范围

本 Skill 提供的估价仅供参考，不等同于最终成交价。实际回收价格可能因商品成色、功能状态、维修记录、配件完整度及实时市场行情而有所不同。

## English

### name

Zhuanzhuan Trade-In Estimator

### short_description

A skill for estimating second-hand item value — upload photos or describe the item to get a trade-in price range.

### long_description

Zhuanzhuan Trade-In Estimator is a smart valuation skill for pre-owned goods. Users can upload photos, type a text description, or combine both. The system identifies the item's category, brand, model, and key attributes to produce a reference trade-in price range.

The skill is designed for a "get an estimate first, decide later" workflow — no commitment to sell is required. Even with incomplete information, it will do its best to return a price range and suggest what details would help refine the quote.

### sample_prompts

- How much is this phone worth?
- Can you estimate the trade-in value of my iPhone 13?
- I'll send a few photos — how much is this laptop worth?
- It's the 128 GB model with minor screen scratches. What's the price range?
- How much could I get for this tablet?
- I'm not sure of the exact model — can you give me a rough estimate from the photos?
